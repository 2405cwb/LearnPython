# Ubuntu更多命令学习
## 创建，写文件
1. touch
* 创建空文件 
  * touch a.txt 
2. cat
* 显示文件内容
  * cat a.txt
* 显示行号
  * cat -n a.txt
* 合并空行显示
  * cat -s a.txt
* 写文件
  * cat > a.txt  
    * **使用ctrl+d**退出
3. tac
* 倒着显示文件内容
  * tac a.txt
* 统计文件中的字节数行数并输出
  * wc a.txt

| d | c |
|---|---|
| b | a |~~
~~
2. 排序
* sort a.txt
3. 忽略或报告重复行
* unqi
  * unqi a.txt
    * -i 
      * 忽略大小写
    * -c 
      * 进行计数
    * -u 
      * 只显示唯一的行
4. 从一个文本文件或者文本流中提取文本列
* cut
  * -d 
    * 后面接分隔符 与-f一起使用
  * -f 
    * 依据-d的分割字符将一段信息分割或分为数段 -f取出第几段的意思
  * -c 
    * 以字符的单位去除固定字符区间
5. 读取文本
* head -n 20 a.txt ***显示前20行***
6. 监控文件变化
* tall -f a.txt
7. 查看命令位置
* which cat
8. 权限设置
* chmod
    * chmod u=rw a.txt 
      * ***三组权限  用户u 组g 其他o***
    * chmod o+wx a.txt 
      * 为其他用户加上wx权限 ***r读取 w写 x运行***
    * chmod 777 a.txt r=4 w=2 x =1  
      * ***7所以如果要赋予rwx权限 则设置为7 77 则为所有用户设置所有权限***
## 快捷键
